Instrucciones

  1. Crea los íconos del proyecto con GIMP.

  2. Crea la base de un sitio web usando una cuenta de Google, GitHub y opcionalmente Visual Sudio Code.

  3. Crear un proyecto de Firebase con el mismo correo que la cuenta de GitHub.

  4. Configura Authorization, Storage y Firestore en tu proyecto de Firebase; añade datos y registra tu administrador.

  5. Edita el código, utilizando las diapositivas que siguen después de esta y pruébalo. Se prueba directamente sobre el sitio web.
